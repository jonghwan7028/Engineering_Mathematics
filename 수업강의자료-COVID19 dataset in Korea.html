1. 라이브러리 가져오기¶

본 분석에 필요한 라이브러리는 Numpy, Pandas, Datetime, Matplotlib, Seaborn, 그리고 Plotly 입니다.

  • Numpy: 파이썬과 함께 데이터의 다양한 연산을 도와주는 필수 라이브러리 입니다.
  • Pandas: 파이썬을 사용하여 데이터를 수집하고 분석 및 가시화할 수 있는 라이브러리 입니다.
  • Datetime: 파이썬에서 날짜와 시간을 연산하는데 유용한 함수를 제공하는 라이브러리 입니다.
  • Matplotlib: 파이썬 결과물을 출력하는 필수 라이브러리 입니다.
  • Seaborn: 파이썬 결과물을 출력하는데 Matplotlib보다 좀더 가시화를 강화한 라이브러리 입니다.
  • Plotly: Seaborn과 같이 Matplotlib보다 좀더 가시화를 강화한 라이브러리 입니다.

2. 데이터 불러오기¶

이번 분석에 사용된 데이터는 Kaggle에 업로드된 Coronavirus-Dataset in South Korea (https://www.kaggle.com/kimjihoo/coronavirusdataset)을 참고하였습니다. 이 데이터셋은 3가지로 구분이 되며, 세부적인 내용은 다음과 같습니다.

  • time.csv: 날짜별 통계현황에 대한 11가지 항목의 데이터
  • route.csv: 환자의 이동경로 관련 6가지 항목의 데이터
  • patient.csv: 환자의 성별 등 14가지 항목(attribute)에 대한 데이터

우선 Pandas 라이브러리를 이용해서 csv 형태의 데이터를 불러오도록 하겠습니다.

4. 일자별 확진자 추세¶

2020년 1월 20일부터 시작된 COVID-19 확진자의 일자별 추세를 한번 살펴보도록 하겠습니다. 이것과 관련된 데이터는 time.csv로 이것을 Pandas를 사용하여 불러 오도록 하겠습니다. time.csv는 11개의 항목이 존재하며, 각각의 항목의 의미는 다음과 같습니다.

  • date: 일자를 의미합니다. 1월 20일부터 3월5일까지 데이터입니다.
  • acc_test : 누적된 검사자
  • acc_negative : 누적 음성 확진자
  • acc_confirmed : 누적 양성 확진자
  • acc_released : 누적 격리해제자
  • acc_deceased : 누적 사망자
  • new_test : 신규 검사자
  • new_negative : 신규 음성 확진자
  • new_confirmend: 신규 양성 확진자
  • new_released : 신규 격리해제자
  • new_deceased : 신규 사망자
